# Kairo.ts

Kairo.ts とは、統合版マインクラフトのアドオン専用のフレームワークで、
複数アドオンを **いい感じに通信させる** ための管理アドオンです。

静空が主体となって制作している**マイクラ人狼 v2.0.0**では、このKairo.tsを
使っていい感じに制作しています。

(そもそもKairo.tsは、マイクラ人狼v2.0.0を作るために静空が自作しました)
※Kairo.tsの発想元はゆたさんです。
