# kairo-js

kairo-js とは、統合版マインクラフトのアドオン専用のフレームワークで、
複数アドオンを **いい感じに通信させる** ための管理アドオンです。

静空が主体となって制作している**マイクラ人狼 v2.0.0**では、このkairo-jsを
使っていい感じに制作しています。

(そもそもkairo-jsは、マイクラ人狼v2.0.0を作るために静空が自作しました)
※アドオン間で通信することができるという ゆたさん の発想から
