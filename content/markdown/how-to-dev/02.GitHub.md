## ssh を通す
GitHubを使う上で、ssh keyを登録しておいた方が何かと便利なので、先に解説しちゃいます。
※仕組み・内容は解説せず、登録方法だけ。

再イン済みのGitHubにアクセスして、右上のアイコンからsettings(設定)の項目へ行きます。
そして、左側の **SSH and GPG key**(SSHとGPG鍵) を開きます。

右上の「New SSH key」を押してください。
**Title**は何を書いても問題ないです。あなた自身が識別可能な好きなタイトルを付けてください。

**Key type**はそのままいじる必要はありません。

そして、**Key**に保存するための、ssh-keyをローカルで生成する必要がります。
Windows環境で、Powershell を起動してください。(多分デフォルトで入ってます)

そこで以下のコマンドを実行しましょう。
`ssh-keygen -t ed25519 -C "your_email@example.com"`
your_email@example.com の部分は、あなたのメールアドレスに置き換えてください。
(メアドを書かなくても大丈夫ではある。)

すると英語でpassphraseを訊かれると思いますが、無視してEnter連打しましょう。
でっかく四角いログが出てきたら成功です。

次に、以下のコマンドを実行します。
`cat ~/.ssh/id_ed25519.pub`
それで出力された1行すべてをコピーし、
GitHubのページに戻り、SSH key を登録するページの、**Key** の欄にそれをペーストしましょう。

そして、「Add SSH key」を押すことで、sshを通すことに成功すると思います。
