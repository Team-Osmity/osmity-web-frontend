# ワールドを開く
以下のdiscord リンクから、マイクラ人狼 v1.4.3のワールドをダウンロードして、
統合版の方へインポートしてください

※現在インポートがバグっているので、.mcworld を .zip へと名称変更し展開して、手動でマイクラのフォルダへ移す必要があります。

## アドオンを適用
ワールドがインポートできた人は、
マイクラ人狼 v1.4.3 のワールドの編集画面へと行き、
ビヘイビアーパックの欄に以下のアドオンがあることを確認してください
```
Kairo
Kairo DataVault
Werewolf GameManager
Werewolf VanillaPack
Werewolf AdditionalRolesI
```
上記のアドオンが存在することが確認できた人は、順にワールドに適用してください

## 動作確認
ワールドを開いて、マイクラ人狼が正常に動作していると成功です！
これで、マイクラ人狼の環境構築はいったん終わりです
