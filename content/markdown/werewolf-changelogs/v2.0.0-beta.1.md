# v2.0.0-beta.1

## 機能追加・不具合修正
-# Features and Bug Fixes

### GameManager
・誰かに殺害された時に、「○○に殺害された」とメッセージを送信するようにしました。
・人狼または大狼の時に、アクションバーで味方のネームタグを表示するようにしました。
・ゲーム開始時に各自の役職に合わせて、役職の説明文のメッセージを送信するようにしました。
・ゲーム終了後の結果表示で、プレイヤーの並び順を役職によってソートするように変更しました。
・プレイヤーが持つアイテムを外に投げ捨てたりできないようにしました。
・「強制終了」のアイテムをゲーム中のインベントリから一時的に削除しました。
